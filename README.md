# Notes App

A full-stack notes application where users can create, edit, and delete notes.

## Tech Stack

- React
- Node.js
- Express
- MongoDB Atlas
- Axios
- Tailwind CSS

## Live Demo

Frontend: https://notes-app-theta-lemon.vercel.app/
Backend API: https://notes-app-dzi9.onrender.com

## Features

- Create notes
- Edit notes
- Delete notes
- Persistent database storage
